# 🚀 SaaSify — Modern SaaS Landing Page

A **premium, conversion-focused SaaS landing page** built with Next.js, featuring animated UI components, dark dashboard previews, testimonials, and pricing tables.

Live Demo: [saas-landing-page-nine-gamma.vercel.app](https://saas-landing-page-nine-gamma.vercel.app)

---

## ⚙️ Tech Stack

| Layer | Technology |
|-------|-----------|
| **Framework** | Next.js 15 (App Router) |
| **Language** | TypeScript |
| **Styling** | Tailwind CSS |
| **Animations** | Framer Motion |
| **Icons** | Lucide React |
| **Deployment** | Vercel |

---

## 💡 Features

* 🎨 Modern dark-themed hero with animated dashboard preview
* 💰 Pricing tables with highlighted recommended plan
* ⭐ Testimonials section with customer reviews
* ⚡ Feature cards with icons and descriptions
* 📱 Fully responsive on all screen sizes
* 🎭 Smooth Framer Motion animations throughout
* 🚀 Deployed on Vercel with CI/CD

---

## 🚀 Getting Started

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/Neric-wizard/saas-landing-page.git
cd saas-landing-page
```

### 2️⃣ Install Dependencies
```bash
npm install
```

### 3️⃣ Run Development Server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it.
